CTBase.jl

The CTBase.jl package is part of the control-toolbox ecosystem.

flowchart TD O(<a href='https://control-toolbox.org/OptimalControl.jl/stable/'>OptimalControl</a>) --> B(<a href='https://control-toolbox.org/OptimalControl.jl/stable/api-ctbase.html'>CTBase</a>) O --> D(<a href='https://control-toolbox.org/OptimalControl.jl/stable/api-ctdirect.html'>CTDirect</a>) O --> F(<a href='https://control-toolbox.org/OptimalControl.jl/stable/api-ctflows.html'>CTFlows</a>) F --> B D --> B style B fill:#FBF275

You may find in this package:

  • Description: tools to manipulate tuples.
  • Differential geometry: tools to compute Lie brackets, Poisson brackets...
  • Exceptions.
  • Initial guess: tools to define an initial guess.
  • Model: tools to modelise an optimal control problem in a functional manner.
  • Parser: tools to modelise an optimal control problem in an abstract manner.
  • Print: check how an optimal control problem is displayed.
  • Repl: control-toolbox repl.
  • Solution: optimal control solution.
  • Types: the common types and associated methods if any.
  • Utils: auxiliary methods.
